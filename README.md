# SCLS - Foundation
## What is SCLS ?
SCLS is an Aster System's set of libraries made to simplify development in C++, with a lot of useful and simple tools made to simplify the life of C++ developers.
## What is SCLS Foundation ?
SCLS Foundation  "Leonhard" is the foundation of the SCLS (and all of the Aster System's tools) library. It contains a lot of basic tools, very simple but quite practical, to develop more easily.
## Requirements
This library is made for C++ 20, but can be used in C++ 17.
It does not need any other requirements.
